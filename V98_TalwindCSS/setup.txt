How to setup Tailwind CSS
Step 1: Run the following commands

npm install -D tailwindcss
npx tailwindcss init
Step 2: Update tailwind.conf.js file to include this line:

content: ["*.html"],
Step 3: create src/input.css to include:

@tailwind base;
@tailwind components;
@tailwind utilities;
Step 4: Include the src/output.css file to your html

Step 5: Run the following command whenever working in tailwind such that output file update hoti rhe automatically:

npx tailwindcss -i ./src/input.css -o ./src/output.css --watch

Step 6: In package.json add above command as name build in scripts to avoid writing such long commands just run npm run build
